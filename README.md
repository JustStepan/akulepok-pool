# Школа плавания - Frontend

Учебный проект на React + Tailwind CSS для школы плавания.

## Технологии

- **React 19** - библиотека для построения UI
- **Vite** - быстрый сборщик и dev-сервер
- **Tailwind CSS** - utility-first CSS фреймворк
- **React Icons** - библиотека иконок

## Структура проекта

```
frontend/
├── src/
│   ├── components/      # Компоненты страницы
│   │   ├── Header.jsx   # Шапка с навигацией
│   │   ├── Hero.jsx     # Главный баннер
│   │   ├── About.jsx    # Секция "О нас"
│   │   ├── Trainers.jsx # Секция "Тренеры"
│   │   ├── News.jsx     # Секция "Новости"
│   │   ├── Pricing.jsx  # Секция "Тарифы"
│   │   ├── Contacts.jsx # Секция "Контакты"
│   │   ├── Placeholder.jsx # Заглушка
│   │   └── Footer.jsx   # Подвал сайта
│   ├── pages/
│   │   └── Admin.jsx    # Админ-панель (визуальная часть)
│   ├── data/
│   │   └── content.js   # Захардкоженные данные
│   ├── App.jsx          # Главный компонент
│   └── index.css        # Стили + Tailwind
├── public/              # Статические файлы
└── package.json         # Зависимости
```

## Установка и запуск

### 1. Установка зависимостей
\`\`\`bash
npm install
\`\`\`

### 2. Запуск dev-сервера
\`\`\`bash
npm run dev
\`\`\`

Сайт будет доступен по адресу: http://localhost:5173/

### 3. Сборка для production
\`\`\`bash
npm run build
\`\`\`

### 4. Предпросмотр production сборки
\`\`\`bash
npm run preview
\`\`\`

## Особенности

### Одностраничное приложение (SPA)
- Все секции на одной странице
- Плавная прокрутка при клике на меню
- Навигация через якорные ссылки (#about, #trainers и т.д.)

### Морская тематика
- Цветовая палитра: голубой, синий, белый
- Кастомные цвета ocean-* (50-900) в Tailwind
- Градиенты для красивых переходов

### Адаптивность
- Мобильная версия (пока частичная)
- Использование Tailwind responsive классов (md:, lg:)

### Компоненты

#### Header
- Фиксированная шапка сайта
- Навигация с плавной прокруткой
- Ссылки на соцсети (VK, Telegram, Instagram, WhatsApp)

#### Hero
- Главный баннер с призывом к действию
- Яркий градиентный фон
- Кнопка "Записаться на занятие"

#### About
- Информация о школе
- Карточки с преимуществами

#### Trainers
- Карточки тренеров
- Аватары (пока заглушки)
- Информация о каждом тренере

#### News
- Список новостей и объявлений
- Форматирование даты
- Заметка о будущем функционале

#### Pricing
- Тарифные планы
- Выделение популярного тарифа
- Кнопки с переходом к контактам

#### Contacts
- Контактная информация
- QR-коды для соцсетей (пока заглушки)
- Иконки от react-icons

#### Placeholder
- Заглушка для будущего функционала
- Идеи для расширения

#### Footer
- Контакты
- Ссылки на соцсети
- Copyright

### Админ-панель (src/pages/Admin.jsx)
Визуальная версия без функционала:
- Управление фоновым изображением
- Добавление/редактирование новостей
- Управление тренерами
- Управление тарифами

## Данные

Все данные сейчас захардкожены в файле `src/data/content.js`:
- Информация о школе
- Тренеры
- Новости
- Тарифы

В будущем эти данные будут приходить от FastAPI backend.
