# Папка для QR-кодов

Сюда нужно поместить изображения QR-кодов для соцсетей:

## Необходимые файлы:

- `vk.png` - QR-код для группы ВКонтакте
- `telegram.png` - QR-код для Telegram канала
- `instagram.png` - QR-код для Instagram
- `whatsapp.png` - QR-код для WhatsApp

## Как создать QR-коды:

### Вариант 1: Онлайн-генераторы
- https://www.qr-code-generator.com/
- https://qr.io/
- https://www.the-qrcode-generator.com/

### Вариант 2: Библиотека (в будущем)
Когда будет админка, можно добавить автоматическую генерацию QR-кодов с помощью библиотеки `qrcode.react`

## Формат файлов:
- **Размер**: желательно 500x500px или больше
- **Формат**: PNG с прозрачным фоном
- **Качество**: высокое, чтобы QR-код хорошо сканировался

## Как добавить файлы:
1. Создайте QR-код для вашей соцсети
2. Скачайте изображение
3. Переименуйте в соответствующее имя (vk.png, telegram.png и т.д.)
4. Поместите в эту папку

После этого QR-коды автоматически появятся на сайте в разделе "Контакты"!
